## Overview
Focus Timer is a simple Chrome extension I built for myself to support focused study sessions.  
I often found it distracting and inconvenient to use my phone just to run a timer while studying, so I decided to create a lightweight, distraction-free timer directly in the browser.

This project is intentionally minimal and designed for personal productivity rather than as a commercial product.

---

## Why I Built This
- I needed a dedicated study timer while working on my laptop
- Using my phone for timing was breaking focus and leading to unnecessary distractions
- I wanted something clean, fast, and always accessible in the browser
- This project helped me practice building a complete Chrome extension from scratch

---

## Features
- Focus and break sessions
- Clean, dark, distraction-free UI
- Start and reset controls
- Session logs with:
  - Total sessions completed
  - Total focus time
  - Calendar-based view of activity
- Designed purely for a personal study workflow

---

## Tech Stack
- HTML
- CSS
- JavaScript
- Chrome Extensions API

---

## Usage
1. Open Chrome and go to `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the project folder
5. Click the extension icon and start a focus session
